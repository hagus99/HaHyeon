Strings in C Language

스트링은 문자열이다. 여러개의 캐릭터들로 이루어져있다
Array of characters
End of string is denoted by null character('\0') *null character 와 null pointer는 다른 말이다. (character와 pointer의 차이)
String vs. Character
char 'H' => 1byte / string "H" => 2bytes(H,\0) / " "(Empty String) => \0
string 과 character arrays의 차이점은 맨뒤에 \0 (null character가 붙는지 안붙는지에 대한 차이)

- string은 그 시작 주소에 의해서 access된다.
- &str[i] == str + i // a substring , starting from the i char.
str[i]번째 부터 뒤의 character가 연결되는 substring이 된다.
- strlen 함수는 순수하게 문자열 길이만 구하며 NULL부분은 포함되지 않는다.
- Note! Assignment to a string is permitted only for initialization.
배열일 때 Assignment는 불가능하지만 포인터이면 Assignment가 가능하다.
ex)
char str1 [9] = "Good Day"; //초기화 할 때는 가능
char str2[9];
char *pStr = NULL;

str1 = "Good Day" //error
str2 = str1 //error
pStr = str1; //가능
