c 언어에서는 기존에 있는 데이터 타입을 조합해서 새로운 데이터 타입을 만들 수 있다.

Typedef
typedef는 사용자가 데이터 타입의 이름을 붙혀서 새로운 데이터 타입을 만들 수 있다.
Syntax: typedef
ex) typedef unsigned int size_t; // in search.h
ex) typedef int INTEGER;
INTEGER x;
ex) typedef char* STRING;
STRING stringPtrArray[20];



Enumerated Type
ex) Colors: RED, BLUE, YELLOW . . .
#define RED 0과 같이 여러개를 선언해줘도 되지만 간단하게 enum을 사용한다.
Syntax: enum tag { identifier_list};
ex) enum Color { RED, BLUE, GREEN, WHITE};
enum Color 자체가 데이터 타입이 된다.
ex) int func(enum Color curColor);

Example
// definition of Color type
enum Color { RED, GREEN, BLUE, WHITE, PURPLE, . . .};
// variable declarations
enum Color x, y, z;

x = BLUE;
y = WHITE;
z = PURPLE;

if(x == BLUE){
. . .
}

switch(y){
case WHITE;
. . .
}

typedef vs. #define
#define INTP int*
INTP pa, pb // same with "int* pa, pb;"

typedef int* INTP;
INTP pa, pb; // same with "int *pa, *pb;"

Structure
Structure: collection of related elements, possibly of different types.
ex) FILE is a structure type defined in stdio.h
Defining a user - defined data type.
struct [tag] {
field list
};

방법 1.
struct STUDENT {
char id[10];
char name[26];
enum Major;
};

struct STUDENT student[50];

방법 2.
typedef struct {
char id[10];
char name [26];
enum Major major;
}STUDENT;

STUDENT student[50];

ex) Accessing Structures
SAMPLE sam2;
printf("x: %d\n",sam2.x);
printf("t: %f\n",sam2.t);
구조체의 요소는 "구조체.요소"의 형식을 써서 접근
구조체는 변수선언, 파라미터 타입, 리턴 타입 등 여러형태로 쓰일 수 있다.
배열이나 스트링은 assignment가 안됐지만 sturcture는 가능하다.

Accessing Structures Through Pointers
*ptr.x = 100; // incorrect
(*ptr).x = 100; // correct
(*pointerName).fieldName = pointerName->fieldName
