문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)가 주어진다.

출력
첫째 줄에 N!을 출력한다.

        
       #include <stdio.h>

    int main()
    {
        int num = 0;

        scanf("%d",&num);
        
        printf("%d",factorial(num));
        
        return 0;
    }

    int factorial(int num){
        if(num==0)
            return 1;
        else 
            return num * factorial(num-1);
        }
